## Prerequisites:

* __Python 3.7__ installation

`pip3 install virtualenv`  
`which python3.7`  
`virtualenv ./ecam_python37_env -p /usr/bin/python3.7`
`source ./AnaProj/ecam_python37_env/bin/activate`
